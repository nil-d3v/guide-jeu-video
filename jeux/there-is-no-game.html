 : 2017

  Description

There Is No Game (littéralement "Il n’y a pas de jeu") commence comme une farce : un narrateur sarcastique vous dit qu’il n’y a rien à faire.

Pourtant, au fil de clics : des puzzles inattendus, du texte rebondissant et un jeu plein de surprises.

Ce jeu est resté un classique des jeux narratifs, bien que vous aurez besoin de quelques bases d'anglais pour résoudre certaines énigmes.
